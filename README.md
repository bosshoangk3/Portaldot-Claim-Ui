# 🚀 Portaldot Claim UI

A modern Web3 interface for claiming POT tokens from the Portaldot ecosystem.

---

## ✨ Overview
This project simulates a token claim (airdrop) experience using a clean and minimal UI.

Built for:
- Demo purposes
- UI/UX showcase
- Web3 front-end practice

---

## ⚙️ Features

- 🔗 Connect Wallet (MetaMask)
- 🎁 Claim Token (Demo Logic)
- 💻 Clean & Responsive UI
- ⚡ Fast interaction (React hooks)

---

## 🛠 Tech Stack

- React
- TailwindCSS
- shadcn/ui
- Ethers.js (ready to integrate)

---

## 📸 Preview

> Simple airdrop claim interface  
> Inspired by real Web3 projects (ZK, LayerZero, Starknet)

---

## 🚧 How It Works

1. Connect your wallet
2. Click "Claim POT"
3. System processes (demo)
4. Receive success status
